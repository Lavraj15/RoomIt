# RoomIt - Meeting Room Booking System

## Tech Stack

- Frontend: Next.js, Tailwind CSS
- Backend: Node.js, Express.js
- Database: MongoDB
- State Management: React Hooks

---

## Features

### Room Management

- View all meeting rooms
- Check room availability
- Book one or more consecutive slots

### Booking Features

- Prevent double booking
- Consecutive slot validation
- Buffer time after booking
- Cancel booking
- Refund / Non-refundable cancellation
- Reschedule booking
- Search bookings by email

### UI Features

- Dark theme
- Responsive design
- Modern cards
- Booking modal
